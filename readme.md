# 💰 Finance Bot

Telegram бот для учета финансов.

## 🚀 Быстрый старт

### Локально:
```bash
pip install -r requirements.txt
python bot.py
```

### На Railway:
1. Скопируйте репозиторий на GitHub
2. На Railway подключите GitHub репозиторий
3. Добавьте переменную окружения `BOT_TOKEN`
4. Бот автоматически запустится

## 📝 Команды

- **/start** - Начать работу с ботом
- **/help** - Справка

## 💳 Использование

**Добавить расход:**
```
еда 500
такси 200
покупка одежда 1500
```

**Добавить доход:**
```
зарплата 50000
доход 10000
фриланс 5000
```

## 🔘 Кнопки

- 📊 Статистика - просмотр расходов
- 📁 Excel отчет - скачать отчет
- ✏️ Редактировать - изменить или удалить транзакцию
- 🗑️ Сбросить данные - удалить все данные
- ℹ️ Справка - справка по использованию

## 🎯 Функции

✅ Учет расходов по категориям
✅ Учет доходов
✅ Статистика по категориям
✅ Экспорт в Excel
✅ Редактирование/удаление транзакций
✅ Ежедневные напоминания (20:00)
✅ Работает на Railway
